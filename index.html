TIKUS DI GOT
Top-Down Adventure Game

Si Tikus tinggal nyaman di got bawah Kota Mentari.

Suatu hari: Dinas Kebersihan akan membersihkan got dalam 8 menit!

Jelajahi got, bantu para penghuni, dan kabur sebelum Petugas datang!

📋 URUTAN MISI
1
Cari Makanan Basi di Got Tengah
2
Berikan ke Pak Koak → Dapat Peta Got
3
Temui Kiki → Dapat Catatan
4
Cari Papan Kayu di Got Utara
5
Berikan ke Nyonya Siput → Dapat Kunci
6
Gunakan kunci → Kabur dari Got!
⚠️ BAHAYA
!
Petugas Got masuk setelah 1 menit — bisa mengejar!
!
Perangkap tersembunyi — injak = Game Over!
!
Hanya 8 menit — kabur sebelum waktu habis!
🎮 PILIH MODE
🖥️ KONTROL PC
WASD / Arrow — Gerak
E — Interaksi / Ambil Item
ESC — Pause / Tutup Dialog
Dekati NPC → tekan E dialog
📱 KONTROL HP
D-Pad — Gerak karakter
Tombol ⚡ — Interaksi / Ambil Item
Tombol ⏸ — Pause
Dekati NPC → tap ⚡ dialog
